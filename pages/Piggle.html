Space Piggle

Unreleased

About The Game

- Made this game for my Games Design and Analysis assignment
- My first ever project in Godot
- Coded using GD script
- All the assets were made inside the Godot engine.

Features

- Collect all the nodes on board the ship to deactivate the drones and be safe
- Avoid getting caught and put back in a cage by the drones
- Navigate the abandoned ship
- Jump on top of drones to destroy them
